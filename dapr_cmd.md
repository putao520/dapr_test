### 启动组件组

```
http模式:
dapr run --app-id myapp --app-port 3500 --dapr-http-port 3500 --components-path ./my-components

grpc模式:
dapr run --app-id myapp --app-port 50001 --app-protocol grpc --dapr-grpc-port 50001 --components-path ./my-components
```
